Hello , I'm Sayani Saha , pursuing MCA from National Institute of Technology , Warangal.

This is my Criminal Identification WebApp. I've used HTML , CSS , JavaScript for the frontend and Python and MySQL at the backend. Face Recognition is implemented via the face-recognition and OpenCV python libraries.

My WebApp scans faces in real-time and checks for a match in the database. One can also add new criminal details in the database.
